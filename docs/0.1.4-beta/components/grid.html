How to layout grids

  • The container class gives its div a specific fixed width. The container-fluid's width changes every time the viewport width changes.
  • The row class is used mainly to hold columns in it.
  • Grid columns are created by specifying the number of 12 available columns you wish to span. For example, three equal columns would use three .col-sm-4

Centering a column

Setting an offset equal to half of the remaining size of the row. For example, a column of size 2 would be centered by adding an offset of 5, that's (12-2)/2. It only works for even column sizes, so only .col-**-2, .col-**-4, col-**-6, col-**-8 and col-**-10 are supported.
