最新版 v0.1.4-beta

はじめに

Rex CSS Frameworkはレスポンシブでモバイル・ファーストな楽天プロジェクトを開発するためのフレームワークです。

RexCSSフレームワークはベータ版であり、現在も開発途中です。

Rex CSSフレームワークベータ版は現在も開発途中ですので不具合を起こす可能性があります。正式版をより良いものにするため、バグなどの不具合にお気づきの場合には、pull requestでのフィードバックをお待ちしております。インストールされる前に、以下を必ずお読みください。

  • こちらはベータ版です。プロダクションでご使用の際には十分ご注意ください。
  • ベータテスト版をご利用中に不具合や要望などがございましたら、Pull Requestを送ってください。
  • npmでpackageをインストールした際には、他のライブラリへの依存関係などを確認の上お使いください。
  • プロジェクトの構成、変数やクラス名は改善のために次期バージョンで変更されることがあります。
  • ベータ版はコアなコンポーネントが含まれますが、時期のリリース版には皆さんのフィードバックを元にコンポーネントを追加していきます。
  • ベータ版の実機テストは、BrowserStack(実際のブラウザの実行画面をWeb経由で表示・操作できるWebサービス)で行っていますが、十分ではありません。何か不具合などにお気づきの場合や、ご意見などがございましたらフィードバックをお待ちしております。

CDN版

すべてのバージョンがcdn版で入手できます。

CDN版のCSSは圧縮されています。完全版とカスタム版があり、プロジェクトの用途に合わせてどちらかをお選びいただけます。個々のコンポーネントのみをCDNでインストールすることも可能です。

a) 完全版：すべてのコンポーネントが含まれます。

b) 完全版：WebFont'NotoSansJP'を含まない（rex-core.css）/WebFont'NotoSansJP'を含む（rex-core-webfonts.css）を以下からお選びください。

NotoSansJPなどのWebFont版を使用する場合には注意が必要です。Google FontsはIE11のサポートをしていません。IE11のサポートが必要な場合には、WebFontのNotoSansJPを含まない（rex-core.css）を使用ください。

カスタム版例：各コンポーネントのセクションで個々のコンポーネントのCDN版の情報をご確認いただけます。

npmでインストール

最新版をnpmでインストールする方法 

npm i rex-css-framework

a) 完全版：すべてのコンポーネントが含まれます。 

/** 
Full version with Web Fonts (Plug and Play) 
JavaScript Import
*/
import 'rex-css-framework/dist/css/rex.css';

b) 完全版：WebFont'NotoSansJP'を含まない（rex-core.css）/WebFont'NotoSansJP'を含む（rex-core-webfonts.css）を以下からお選びください。

NotoSansJPなどのWebFont版を使用する場合には注意が必要です。Google FontsはIE11のサポートをしていません。IE11のサポートが必要な場合には、WebFontのNotoSansJPを含まない（rex-core.css）を使用ください。 
/** 
Core (Default with Font Family)
JavaScript Import
*/
import 'rex-css-framework/dist/css/rex-core.css';
//or 
/** 
Core (Alternative with Web Fonts)
JavaScript Import
*/
import 'rex-css-framework/dist/css/rex-core-webfonts.css';

カスタム版例：各コンポーネントのセクションで個々のコンポーネントのCDN版の情報をご確認いただけます。 

/** 
Core (Default with Font Family)
JavaScript Import
*/
import 'rex-css-framework/dist/css/rex-core.css';
import 'rex-css-framework/dist/css/rex-core-typography.css';

使われている技術

Rex Css Frameworkのベータ版は、Bootstrapv3をベースにRex Designを導入して作られています。
CSSはSASS(SCSS)で書かれています。配布用のbuildファイルには完全版'rex.css'とコンポーネントごとのCSS、例えばタイポグラフィのみのCSS'rex-typography.css' などがあります。 CSSをプロジェクトで利用するには、CDN版を使うかインストール後にdist/ディレクトリ以下のCSSファイルをお使いください。
ベータ版はCSSのみが含まれます。現バージョンではJavaScriptは含まれません。
次のStableバージョンではインタラクションを含むReact UI Componet版をリリース予定です。こちらはBootStrap4をベースに作られる予定です。
